What is HTML?
HTML stands for Hyper Text Markup Language. HTML describes the structure of Web pages using markup HTML elements are the building blocks of HTML pages HTML elements are represented by tags

CSS3 Styling

CSS stands for Cascading Style Sheets. CSS describes how HTML elements are suppose to be displayed on screen.CSS is a language that describes the style of an HTML document. CSS describes how HTML elements should be displayed.

JavaScript

JavaScript is one of the 3 languages all web developers must learn. HTML to define the content of web pages. CSS to specify the layout of web pages. JavaScript to program the behavior of web pages.JavaScript is easy to learn.
